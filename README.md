# Sistem Manajemen Keuangan Anak Kos
**Sistem Manajemen Keuangan Anak Kos** adalah aplikasi manajemen keuangan berbasis web yang dirancang khusus untuk mahasiswa dan penghuni kos. Aplikasi ini membantu pengguna mencatat pemasukan dan pengeluaran harian dengan antarmuka yang modern dan pengalaman pengguna yang interaktif.
## Fitur Utama

* **Autentikasi Aman:** Sistem Login dan Register dengan enkripsi password.
* **Dashboard Interaktif:** Ringkasan saldo, pemasukan, dan pengeluaran secara *real-time*.
* **CRUD Transaksi:** Tambah, Baca, Edit, dan Hapus transaksi dengan mudah.
* **Visualisasi Data:** Grafik Donat (*Doughnut Chart*) untuk melihat persentase pengeluaran vs pemasukan.
* **Ekspor Laporan:** Unduh riwayat transaksi ke format CSV (Excel).
* **Personalisasi:**
    * **Quick Notes:** Catatan tersimpan otomatis di browser (Local Storage).
    * **Cover:** Ganti gambar sampul sesuai yang diignikan (disimpan per user).
